# Hotel Priscilla Booking

Reservation system for a specific hotel.

## Description
This project is a web system for managing rooms, guests, and bookings. Implemented in pure Java (backend) with MySQL connection.

## Project Structure
- `/src` → Java source code
- `/lib` → External libraries (MySQL Connector, etc.)
- `.gitignore` → Ignored files

## Main Entities
- **Room** → Hotel rooms
- **Guest** → Registered guests
- **Booking** → Made reservations

## Dependencies
- JDK 17+
- MySQL / XAMPP
- External libraries in `/lib`

## How to Run
1. Open the project in IntelliJ IDEA
2. Configure the database connection in `DatabaseConnection.java`
3. Run the main class
